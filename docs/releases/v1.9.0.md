> Introduces parallel NetCDF-4/HDF5 I/O capabilities with MPI, including CI infrastructure, parallel build system support, and comprehensive C/Fortran example programs demonstrating collective parallel I/O patterns.

## Features

### Parallel I/O Examples

**C Example** (`examples/parallelIO/square16_par.c`):
- Opens NetCDF-4 file with `nc_create_par()` using `NC_MPIIO` flag
- Enables collective I/O via `nc_var_par_access(ncid, varid, NC_COLLECTIVE)`
- Computes per-rank hyperslab offsets for 2√ó2 quadrant decomposition
- Writes 8√ó8 arrays with `nc_put_vara_int()` and verifies with parallel read-back
- Validates data integrity across all ranks

**Fortran Example** (`examples/parallelIO/f_square16_par.f90`):
- Equivalent functionality using `nf90_create_par()` and `nf90_var_par_access()`
- Demonstrates 1-based indexing for Fortran hyperslab calculations
- Column-major storage considerations documented

**Learning Objectives Documented**:
- Parallel file creation and collective I/O modes
- Domain decomposition patterns for scientific computing
- Hyperslab operations with `start[]`/`count[]` arrays
- Error handling in MPI environments

### Build System Support

**CMake**:
- `ENABLE_PARALLEL_TESTS` option (default OFF)
- `MPIEXEC_EXECUTABLE` configuration for test execution
- Links against parallel-enabled NetCDF-C with `NC_HAS_PARALLEL4` detection

**Autotools**:
- `--enable-parallel-tests` configure flag
- `--with-mpiexec=PATH` for specifying MPI launcher
- Detects parallel NetCDF-C via `netcdf_meta.h` macros

### CI/CD Infrastructure

**Parallel Build Matrix** (`ci-parallel.yml`):
- Matrix: CMake √ó Autotools √ó OpenMPI (2 jobs)
- Builds HDF5, NetCDF-C, and NetCDF-Fortran with parallel support
- Cached dependencies with parallel-specific cache keys
- Runs full test suite including parallel examples with 4 MPI processes
- OpenMPI root execution workarounds for containerized environments

## Build System

### New Configuration Options

**CMake**:
```cmake
-DENABLE_PARALLEL_TESTS=ON          # Enable parallel I/O examples and tests
-DMPIEXEC_EXECUTABLE=/path/to/mpirun # Specify MPI launcher
```

**Autotools**:
```bash
./configure --enable-parallel-tests --with-mpiexec=/usr/bin/mpirun
```

### Dependency Requirements

Parallel builds require:
- MPI implementation (OpenMPI, MPICH, or Intel MPI)
- HDF5 built with parallel support (`-DHDF5_ENABLE_PARALLEL=ON`)
- NetCDF-C built with `--enable-parallel-netcdf4`
- NetCDF-Fortran built with MPI compilers
