### Documentation System
- **Build Systems**: Full support in both CMake and Autotools
  - CMake: `make docs` target with `-DBUILD_DOCUMENTATION=ON/OFF`
  - Autotools: `make docs` target with `--enable/disable-docs`
- **Configuration**: `docs/Doxyfile.in` with strict warning settings (`WARN_AS_ERROR = YES`)
- **Output**: Professional HTML documentation in `docs/html/`

### Source Code Documentation
- All functions in `src/` directory have complete Doxygen comments
- Public API headers (`include/*.h`) fully documented
- Compression plugin APIs (`ncsqueeze.h`) documented with usage examples
