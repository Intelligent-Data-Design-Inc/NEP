# v1.5.1: Add NetCDF Example Programs

**Labels:** enhancement, v1.5.1, epic

## Overview
Integrate comprehensive NetCDF example programs into NEP repository to provide users with practical demonstrations of NetCDF API usage. Examples cover classic NetCDF, NetCDF-4, and Fortran interfaces.

## Goals
- Provide working examples of NetCDF programming patterns
- Demonstrate both C and Fortran APIs
- Include examples in build system with option to disable
- Run examples as automated tests with output validation
- Integrate examples into Doxygen documentation

## Sprints

### Sprint 1: Add Examples Directory and Build Integration
**Issue:** See `v1.5.1-sprint-1.md`

**Deliverables:**
- Examples directory with all programs copied from source
- CMake and Autotools build integration

- Doxygen documentation updated

## Example Programs Included

### C Classic Examples (6 programs)
- `simple_2D.c` - Basic 2D array example
- `coord_vars.c` - Coordinate variables
- `format_variants.c` - Different NetCDF formats
- `size_limits.c` - Size and dimension limits
- `unlimited_dim.c` - Unlimited dimensions
- `var4d.c` - 4D variable example

### C NetCDF-4 Examples (5 programs)
- `simple_nc4.c` - Basic NetCDF-4 example
- `compression.c` - Compression features
- `chunking_performance.c` - Chunking and performance
- `multi_unlimited.c` - Multiple unlimited dimensions
- `user_types.c` - User-defined types

### Fortran Classic Examples (6 programs)

- `f_format_variants.f90`

### Fortran NetCDF-4 Examples (5 programs)

## Build Configuration
- CMake: `-DBUILD_EXAMPLES=ON/OFF` (default: ON)
- Autotools: `--enable-examples/--disable-examples` (default: enabled)
- Fortran examples require `ENABLE_FORTRAN=ON` and netcdf-fortran library
