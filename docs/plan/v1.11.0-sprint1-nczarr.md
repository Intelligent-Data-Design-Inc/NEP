# V1.11.0 Sprint 1: Add NcZarr Example

**Objective**: Create `examples/nczarr/nczarr_simple.c` demonstrating creation, metadata, writing, and reading of a local NcZarr dataset using the NetCDF-C API. The example is gated by NetCDF-C NcZarr support (`NC_HAS_NCZARR`) and is built only when examples are enabled.

---

## Background

NcZarr is the NetCDF Zarr cloud format introduced in netCDF-C 4.8.0. It stores a netCDF-4 enhanced data model in a Zarr directory tree, allowing the same `nc_create()`, `nc_open()`, `nc_def_dim()`, `nc_def_var()`, and data-access APIs to target local filesystem Zarr stores or remote object storage. For Sprint 1 we focus on local filesystem storage, which avoids network configuration and keeps the example self-contained and CI-friendly.

**Storage URL**: `file://simple_nczarr.zarr#mode=nczarr`
- `file://` selects the local filesystem Zarr storage backend.
- `#mode=nczarr` activates the NcZarr dispatch layer.
- The resulting dataset is a directory named `simple_nczarr.zarr`.

**Dataset**: 2D temperature on a 4×5 y×x grid, `NC_FLOAT`, with a `units` attribute.

## Tasks

### 1. Create `examples/nczarr/nczarr_simple.c`

**File**: `examples/nczarr/nczarr_simple.c`

Create a minimal, self-contained NcZarr example that follows the NEP example style: the `if ((retval = nc_XXX())) ERR(retval);` error-handling pattern, compact print statements, and no decorative headers.

```c
/* nczarr_simple.c - Demonstrate local NcZarr create, metadata, write, and read.
 * Edward Hartnett, Intelligent Data Design, Inc. */

#include <netcdf.h>
#include <stdio.h>
#include <stdlib.h>

#define FILE_URL "file://simple_nczarr.zarr#mode=nczarr"
#define NY 4
#define NX 5
#define NDIMS 2

#define ERR(e) do { \
    if (e) { \
        fprintf(stderr, "Error: %s at line %d\n", nc_strerror(e), __LINE__); \
        return 1; \
    } \
} while(0)

int main(void)
{
    int ncid, dimids[NDIMS], varid, retval;
    size_t i, j;
    float data[NY][NX], data_in[NY][NX];

    /* Generate simple 2D temperature field */
    for (j = 0; j < NY; j++)
        for (i = 0; i < NX; i++)
            data[j][i] = 280.0f + (float)j * 2.0f + (float)i * 0.5f;

    /* Create local NcZarr dataset */
    if ((retval = nc_create(FILE_URL, NC_CLOBBER | NC_NETCDF4, &ncid)))
        ERR(retval);

    if ((retval = nc_def_dim(ncid, "y", NY, &dimids[0])))
        ERR(retval);
    if ((retval = nc_def_dim(ncid, "x", NX, &dimids[1])))
        ERR(retval);

    if ((retval = nc_def_var(ncid, "temperature", NC_FLOAT, NDIMS, dimids, &varid)))
        ERR(retval);
    if ((retval = nc_put_att_text(ncid, varid, "units", 1, "K")))
        ERR(retval);
    if ((retval = nc_put_att_text(ncid, varid, "long_name", 11, "Temperature")))
        ERR(retval);

    if ((retval = nc_enddef(ncid)))
        ERR(retval);
    if ((retval = nc_put_var_float(ncid, varid, &data[0][0])))
        ERR(retval);
    if ((retval = nc_close(ncid)))
        ERR(retval);

    printf("Created %s\n", FILE_URL);

    /* Reopen and validate metadata */
    int ndims_in, nvars_in;
    size_t ylen, xlen;

    if ((retval = nc_open(FILE_URL, NC_NOWRITE, &ncid)))
        ERR(retval);
    if ((retval = nc_inq(ncid, &ndims_in, &nvars_in, NULL, NULL)))
        ERR(retval);
    if ((retval = nc_inq_dimlen(ncid, dimids[0], &ylen)))
        ERR(retval);
    if ((retval = nc_inq_dimlen(ncid, dimids[1], &xlen)))
        ERR(retval);

    printf("Dataset: %d dims, %d vars, y=%zu, x=%zu\n", ndims_in, nvars_in, ylen, xlen);

    if ((retval = nc_inq_varid(ncid, "temperature", &varid)))
        ERR(retval);
    if ((retval = nc_get_var_float(ncid, varid, &data_in[0][0])))
        ERR(retval);
    if ((retval = nc_close(ncid)))
        ERR(retval);

    /* Verify data */
    for (j = 0; j < NY; j++) {
        for (i = 0; i < NX; i++) {
            if (data_in[j][i] != data[j][i]) {
                fprintf(stderr, "Mismatch at %zu,%zu: got %f, expected %f\n",
                        j, i, data_in[j][i], data[j][i]);
                return 1;
            }
        }
    }

    printf("Read %d values, all correct. Done.\n", NY * NX);
    return 0;
}
```

### 4. Create `examples/nczarr/CMakeLists.txt`

**File**: `examples/nczarr/CMakeLists.txt`

```cmake
# CMakeLists.txt for NcZarr examples
# Edward Hartnett, Intelligent Data Design, Inc.

# Find nc-config from the NetCDF library directories
find_program(NC_CONFIG nc-config
    HINTS ${NETCDF_LIBRARY_DIRS}/../bin
    PATHS ${NETCDF_LIBRARY_DIRS}/../bin
)

if(NOT NC_CONFIG)
    message(FATAL_ERROR "nc-config not found. Cannot build examples.")
endif()

# Get NetCDF compile and link flags using nc-config
execute_process(
    COMMAND ${NC_CONFIG} --cflags
    OUTPUT_VARIABLE NC_CFLAGS
    OUTPUT_STRIP_TRAILING_WHITESPACE
)
execute_process(
    COMMAND ${NC_CONFIG} --libs
    OUTPUT_VARIABLE NC_LIBS
    OUTPUT_STRIP_TRAILING_WHITESPACE
)

separate_arguments(NC_CFLAGS_LIST UNIX_COMMAND "${NC_CFLAGS}")
separate_arguments(NC_LIBS_LIST UNIX_COMMAND "${NC_LIBS}")

get_filename_component(HDF5_LIB_DIR "${HDF5_INCLUDE_DIR}/../lib" ABSOLUTE)
execute_process(
    COMMAND ${NC_CONFIG} --libdir
    OUTPUT_VARIABLE NC_LIBDIR
    OUTPUT_STRIP_TRAILING_WHITESPACE
)
set(_nep_ld_env "LD_LIBRARY_PATH=${HDF5_LIB_DIR}:${NC_LIBDIR}:$ENV{LD_LIBRARY_PATH}")

add_executable(nczarr_simple nczarr_simple.c)
target_compile_options(nczarr_simple PRIVATE ${NC_CFLAGS_LIST})
target_link_directories(nczarr_simple PRIVATE ${HDF5_LIB_DIR})
target_link_libraries(nczarr_simple PRIVATE ${NC_LIBS_LIST} hdf5 hdf5_hl)

if(BUILD_TESTING)
    add_test(NAME nczarr_simple COMMAND nczarr_simple)
    set_tests_properties(nczarr_simple PROPERTIES ENVIRONMENT "${_nep_ld_env}")
endif()

# Configure validation script with ncdump path from top-level CMakeLists.txt
configure_file(${CMAKE_CURRENT_SOURCE_DIR}/../validate_cdl.sh.in
               ${CMAKE_CURRENT_BINARY_DIR}/validate_cdl.sh
               @ONLY)

if(BUILD_TESTING)
    add_test(NAME nczarr_simple_validate
             COMMAND bash -c "./nczarr_simple && bash validate_cdl.sh nczarr_simple 'file://simple_nczarr.zarr#mode=nczarr' ${CMAKE_CURRENT_SOURCE_DIR}/../expected_output/nczarr_simple_expected.cdl"
             WORKING_DIRECTORY ${CMAKE_CURRENT_BINARY_DIR})
    set_tests_properties(nczarr_simple_validate PROPERTIES
                         DEPENDS nczarr_simple
                         ENVIRONMENT "${_nep_ld_env}")
endif()
```

## Notes

- **Local filesystem only**: The URL `file://simple_nczarr.zarr#mode=nczarr` avoids network dependencies and keeps CI reliable.
- **Build gating**: The example is only built when `NC_HAS_NCZARR` is present; otherwise the directory is skipped silently.
- **Cleanup**: NcZarr produces a directory tree, so `clean-local` removes `simple_nczarr.zarr` rather than relying on `CLEANFILES`.
- **CDL stability**: If `ncdump` output contains runtime-specific NcZarr metadata, skip CDL diff and rely on the example's internal validation.
- **No Fortran example in Sprint 1**: Fortran support can be added in a later sprint once the C example is established.
