# V1.9.0 Sprint 2: Configure/CMake Changes for Parallel I/O

**Objective**: Add build system support for parallel I/O testing including `--enable-parallel-tests` and `--with-mpiexec` options (Autotools) and `ENABLE_PARALLEL_TESTS` (CMake), with automatic detection of parallel-enabled NetCDF-C via `netcdf_meta.h`.

---

## Background

Sprint 1 established CI builds with MPI compilers. Sprint 2 adds the build system infrastructure needed for parallel I/O tests. This includes:
- Options to enable/disable parallel test programs
- Configuration for which `mpiexec`/`mpirun` command to use
- Detection of parallel-enabled NetCDF-C (required for parallel I/O)
- Clear configure output showing what was found

No parallel test programs are added in this sprint â€” those come in Sprint 3. This sprint only prepares the build system to support them.

- **Sprint 1 (Complete)**: MPI compiler CI workflow with `mpicc`/`mpif90`

## Tasks

### 1. Autotools: Add `--enable-parallel-tests` and `--with-mpiexec`

**File**: `configure.ac`

**Add new configure options**:
```autoconf
AC_ARG_ENABLE([parallel-tests],
    [AS_HELP_STRING([--enable-parallel-tests],
        [Enable parallel I/O test programs (default: disabled)])],
    [enable_parallel_tests=$enableval],
    [enable_parallel_tests=no])

AC_ARG_WITH([mpiexec],
    [AS_HELP_STRING([--with-mpiexec=PATH],
        [Specify path to mpiexec/mpirun for parallel tests])],
    [MPIEXEC_PATH=$withval],
    [MPIEXEC_PATH=auto])
```

**Add detection logic** (after NetCDF-C is found):
```autoconf
# Check for parallel NetCDF-C if parallel tests enabled
if test "x$enable_parallel_tests" = xyes; then
    # Check netcdf_meta.h for parallel support
    AC_CHECK_DECLS([NC_HAS_PARALLEL4], [], [], [[#include <netcdf_meta.h>]])
    
    if test "x$ac_cv_have_decl_NC_HAS_PARALLEL4" != xyes; then
        AC_MSG_ERROR([--enable-parallel-tests requires NetCDF-C built with parallel support (NC_HAS_PARALLEL4 not found)])
    fi
    
    # Find mpiexec
    if test "x$MPIEXEC_PATH" = xauto; then
        AC_PATH_PROG([MPIEXEC], [mpiexec], [none])
        if test "x$MPIEXEC" = xnone; then
            AC_PATH_PROG([MPIEXEC], [mpirun], [none])
        fi
        if test "x$MPIEXEC" = xnone; then
            AC_PATH_PROG([MPIEXEC], [srun], [none])
        fi
    else
        MPIEXEC=$MPIEXEC_PATH
        AC_MSG_CHECKING([for mpiexec at specified path])
        if test -x "$MPIEXEC"; then
            AC_MSG_RESULT([yes])
        else
            AC_MSG_RESULT([no])
            AC_MSG_ERROR([--with-mpiexec=$MPIEXEC_PATH does not exist or is not executable])
        fi
    fi
    
    if test "x$MPIEXEC" = xnone; then
        AC_MSG_ERROR([--enable-parallel-tests requires mpiexec/mpirun/srun (use --with-mpiexec=PATH to specify)])
    fi
    
    AC_DEFINE([HAVE_PARALLEL_TESTS], [1], [Parallel I/O tests enabled])
fi

AM_CONDITIONAL([HAVE_PARALLEL_TESTS], [test "x$enable_parallel_tests" = xyes])
AC_SUBST([MPIEXEC])
```

**Add config.h.in entries**:
```autoconf
#undef HAVE_PARALLEL_TESTS
#undef MPIEXEC
```

**Update configure summary output** (add at end):
```autoconf
echo ""
echo "Parallel I/O:"
echo "  Parallel tests enabled: $enable_parallel_tests"
if test "x$enable_parallel_tests" = xyes; then
    echo "  Parallel NetCDF-C: yes (NC_HAS_PARALLEL4)"
    echo "  mpiexec: $MPIEXEC"
fi
```

### 3. CMake: Add `ENABLE_PARALLEL_TESTS` option

**File**: `CMakeLists.txt`

**Add option** (near other ENABLE_* options):
```cmake
option(ENABLE_PARALLEL_TESTS "Enable parallel I/O test programs" OFF)
```

**Add detection logic** (after NetCDF-C is found):
```cmake
if(ENABLE_PARALLEL_TESTS)
    # Check for parallel NetCDF-C
    include(CheckSymbolExists)
    check_symbol_exists(NC_HAS_PARALLEL4 "netcdf_meta.h" HAVE_PARALLEL_NETCDF)
    
    if(NOT HAVE_PARALLEL_NETCDF)
        message(FATAL_ERROR "ENABLE_PARALLEL_TESTS requires NetCDF-C built with parallel support (NC_HAS_PARALLEL4 not found)")
    endif()
    
    # Find mpiexec
    if(NOT MPIEXEC_EXECUTABLE)
        find_program(MPIEXEC_EXECUTABLE 
            NAMES mpiexec mpirun srun
            DOC "MPI execution command for parallel tests"
        )
    endif()
    
    if(NOT MPIEXEC_EXECUTABLE)
        message(FATAL_ERROR "ENABLE_PARALLEL_TESTS requires mpiexec/mpirun/srun (set MPIEXEC_EXECUTABLE to specify)")
    endif()
    
    set(HAVE_PARALLEL_TESTS ON)
    message(STATUS "Parallel NetCDF-C: Found (NC_HAS_PARALLEL4)")
    message(STATUS "mpiexec: ${MPIEXEC_EXECUTABLE}")
endif()
```

**Add to config.h.cmake.in**:

### 5. Update CI workflow to enable parallel tests

**File**: `.github/workflows/ci-parallel.yml`

**Update CMake configure** to enable parallel tests:
```yaml
- name: Configure with CMake (MPI)
  run: |
    mkdir -p build && cd build
    cmake .. \
      -DCMAKE_C_COMPILER=mpicc \
      -DCMAKE_Fortran_COMPILER=mpif90 \
      -DCMAKE_C_FLAGS="-Wall -Werror" \
      -DCMAKE_Fortran_FLAGS="-Wall -Werror" \
      -DCMAKE_PREFIX_PATH="$GITHUB_WORKSPACE/hdf5-install;$GITHUB_WORKSPACE/netcdf-c-install;$GITHUB_WORKSPACE/netcdf-fortran-install" \
      -DENABLE_FORTRAN=ON \
      -DENABLE_PARALLEL_TESTS=ON \
      -DENABLE_GRIB2=OFF \
      -DENABLE_CDF=OFF \
      -DENABLE_GEOTIFF=OFF \
      -DBUILD_LZ4=OFF \
      -DBUILD_BZIP2=OFF \
      -DBUILD_DOCUMENTATION=OFF \
      -DBUILD_EXAMPLES=OFF \
      -DBUILD_TESTING=ON
```

**Update Autotools configure**:
```yaml
- name: Configure with Autotools (MPI)
  run: |
    autoreconf -fiv
    export PATH="$GITHUB_WORKSPACE/hdf5-install/bin:$PATH"
    export CPPFLAGS="-I$GITHUB_WORKSPACE/hdf5-install/include -I$GITHUB_WORKSPACE/netcdf-c-install/include -I$GITHUB_WORKSPACE/netcdf-fortran-install/include"
    export LDFLAGS="-L$GITHUB_WORKSPACE/hdf5-install/lib -L$GITHUB_WORKSPACE/netcdf-c-install/lib -L$GITHUB_WORKSPACE/netcdf-fortran-install/lib"
    export LD_LIBRARY_PATH="$GITHUB_WORKSPACE/hdf5-install/lib:$GITHUB_WORKSPACE/netcdf-c-install/lib:$GITHUB_WORKSPACE/netcdf-fortran-install/lib:$LD_LIBRARY_PATH"
    CC=mpicc FC=mpif90 ./configure \
      --with-hdf5=$GITHUB_WORKSPACE/hdf5-install/bin/h5cc \
      --enable-fortran \
      --enable-parallel-tests \
      --with-mpiexec=/usr/bin/mpiexec \
      --disable-grib2 \
      --disable-cdf \
      --disable-geotiff \
      --disable-lz4 \
      --disable-bzip2 \
      --disable-docs \
      --disable-examples \
      || (echo "Configure failed:"; cat config.log; exit 1)
```

Note: Use `/usr/bin/mpiexec` for OpenMPI, `/usr/bin/mpirun.mpich` for MPICH (adjust per matrix).
