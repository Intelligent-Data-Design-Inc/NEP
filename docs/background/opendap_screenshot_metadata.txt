title: OPeNDAP DAP2 Data Access Form
caption: The OPeNDAP DAP2 Data Access Form provides an interactive web interface for exploring remote datasets hosted on Hyrax servers. Users can view dataset metadata, inspect variable dimensions and attributes, and download data in multiple formats including ASCII, CoverageJSON, NetCDF-3, NetCDF-4, and binary DAP. The form displays the sst.mnmean.nc sea surface temperature dataset with dimensions time=0..1856, lat=0..88, lon=0..179, along with coordinate variables lat, lon, time, and time_bnds. Constraint expressions can be entered in variable text boxes to subset data server-side before transmission.
