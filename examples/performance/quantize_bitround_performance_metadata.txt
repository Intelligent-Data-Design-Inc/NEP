title: NetCDF-4 Quantization + Compression: BitRound

caption: Results from the quantize.c example program in the NetCDF Expansion Pack. Dataset is a 500x180x360 NC_FLOAT temperature array (~129 MB), stored with 10x45x90 chunks. BitRound quantization at NSB values {3,6,9,13,16,19,23} combined with five lossless filters (shuffle=on). X-axis remapped to NSD equivalents (NSB 3=NSD 1 through NSB 23=NSD 7). Top chart: compression ratio. Middle chart: write time. Bottom chart: maximum absolute error in Kelvin.

alt_text: Three vertically stacked line charts with x-axis of NSD-equivalent values 1 through 7 (remapped from BitRound NSB values). Five lines per chart represent deflate, zstandard, szip, lz4, and bzip2 filters, distinguished by line style and marker. Top chart: compression ratio with dotted reference lines for lossless-only ratios. Middle chart: write time in seconds. Bottom chart: maximum absolute error in Kelvin decreasing with NSD.
